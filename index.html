Portafolio

Rick and Morty API

Este proyecto fue creado con React.js cuyo propósito es consumir una API publica que en este caso es la API de Rick and Morty. Se implementó el uso de los hooks de React, el uso de un scroll infinito, la implementación de una sección de favoritos, entre otras cosas.

Pokédex (Android)

Este proyecto fue desarrollado en React Native como aplicación móvil, Su funcionalidad es presentar en pantalla todos los Pokemones existentes, permitiéndonos ver sus estadísticas, la aplicación cuenta con la opción de agregar a favoritos a los pokemones que uno desee.

Cotiza Criptomonedas

En este proyecto se realiza el cálculo del valor de las Criptomonedas más populares en tiempo real con el uso de React.js y gracias al uso de una API que proporciona información de la gran mayoría de las Criptomonedas, se puede consultar en Dólar, Peso, Euro y Libra.

Pacientes veterinaria

Este proyecto tiene la finalidad de ayudar con el seguimiento de los pacientes que pueda tener una veterinaria, con el uso de React.js y local storage se implementaron formularios para guardar y hacer modificaciones de los pacientes a atender.
